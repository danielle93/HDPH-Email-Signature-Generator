## Adding New Employee Headshots & Cu

Sign in to Airtable.com<br>
Go to `https://airtable.com/appQfVniek9Vq2z2e`<br>
Select either the Employee Images table or Culture Images table<br>
Make sure you are in Gallery View<br>
Drag and drop images from folder down to the blue circle with the plus icon on the bottom right of Airtable<br>
Click "Create X Records"<br>
Culture images need to be 800px wide and optimized<br>
Employee headshots need to have the naming convention firstlast.png (ex: johnsmith.png)<br><br>
You can view tokens here, the keys are located in 1pass `https://airtable.com/create/tokens`
